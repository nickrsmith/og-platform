# Contributing

Thank you for your interest in contributing to the O&G Platform!

## Development Setup

1. Follow the [Setup Guide](./SETUP_GUIDE.md) to get the development environment running
2. Read service-specific README files for detailed setup instructions

## Code Style

- **TypeScript/JavaScript**: Follow ESLint configuration in each service
- **Go**: Follow Go standard formatting (`gofmt`)
- **React/Vue**: Follow framework best practices and existing patterns

## Testing

- Write tests for new features
- Ensure existing tests pass
- Aim for good test coverage

## Pull Requests

1. Create a feature branch from `main`
2. Make your changes
3. Ensure tests pass
4. Update documentation if needed
5. Submit a pull request with a clear description

## Questions?

Please open an issue for questions or discussions.

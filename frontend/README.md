# Frontend Applications

This folder contains all frontend applications for the O&G Platform.

## Applications

- **og-marketplace/** - Vue 3 marketplace frontend
- **og-data-room-frontend/** - React data room frontend
- **og-dashboard/** - Dashboard component (to be built)

## Structure

Each application is a separate repository with its own:
- Package/dependency management
- Build configuration
- Docker setup
- Documentation

## Development

See individual application README files for setup and development instructions.
